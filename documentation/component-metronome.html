Metronome Processor

The Metronome processor provides a simple metronome function. The controls are as follows:

Starts and stops the metronome playing.

The measure the metronome is set to play at can be altered by clicking the numbers here and entering new values.

As with the File Player processor, this links the metronome's play button to the main transport's play button. Meaning it will start playing when the main transport starts playing.

Click Sound

By default the metronome plays a simple sine tone as its click, with a higher pitched tone marking the start of the bar. This can be overridden by opening the Metronome's editor:

Here, the accent and click sounds can be replaced with sound files. The available sound file formats are the same as with the File Player processor.
